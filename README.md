# cs2-karambit-p4-checker
A lightweight desktop tool to check rare paint seeds for CS2 Karambit Doppler Phase 4.
# 🔪 CS2 爪子刀多普勒 P4 稀有模板检测工具
以后有空在更新蝴蝶和m9的暂时买不起
这是一个为 CS2 玩家设计的轻量级桌面程序，用于快速检测 **爪子刀 | 多普勒 (阶段 4)** 的稀有图案编号 (Paint Seed)。

## 🏆 功能亮点
- **一键检测**：输入模板号，自动识别 T1/T2/T3 等级。
- **详细描述**：显示“100色”、“无明显白斑断层”等具体稀有特征。
- **自动清空**：检测普通模板自动清空输入框，稀有模板自动保留，方便连续查询。
- **排行参考**：右侧内置排行图鉴，无需切屏查看。

## 🚀 下载使用
请在页面右侧的 [Releases](https://github.com/RsRensan/cs2-karambit-p4-checker/releases) 区域下载最新的 `karambit_check_p4.exe` 即可运行。

## 🛠️ 开发说明
- **语言**：Python 3.13
- **界面**：Tkinter GUI
- **声明**：本工具数据仅针对爪子刀多普勒 P4 模板逻辑。
